📚 Guia de Funcionalidades - Preço-Hub

✅ Novas funcionalidades implementadas!

Descubra como usar cada recurso do sistema de comparação de preços.

🔍 1. Buscador de Produtos

O que é: Sistema de busca avançado para encontrar produtos rapidamente.

Como usar:
  • Acesse Buscar na navbar
  • Digite o nome do produto (mín. 2 caracteres)
  • Refine com filtros: Categoria, Mercado, Faixa de Preço
  • Clique no botão 🔍 Buscar
  • Compare preços e adicione à lista

💡 Dica: Os produtos com melhor preço aparecem destacados em verde!

📊 2. Comparador Visual

O que é: Exibe todos os produtos com comparação visual lado a lado e gráficos.

Como usar:
  • Acesse Comparador na navbar
  • Veja o resumo de economia no topo (total, máxima, média)
  • Analise os gráficos interativos
  • Consulte a tabela comparativa com todos os preços

💡 Dica: Os gráficos mostram em verde o melhor preço em cada mercado!

💰 3. Cálculo Automático de Economia

O que é: Sistema que calcula automaticamente quanto você economiza escolhendo o melhor preço.

Como funciona:
  • Economia por produto: Diferença entre o preço máximo e mínimo
  • Economia máxima: Soma de todas as economias possíveis
  • Economia média: Economia média por produto
  • Badges mostram economia de cada item na busca

💡 Exemplo: Arroz custa R$ 29,90 (Savegnago), R$ 31,50 (Favetta), R$ 30,20 (Pague Menos) = Economia de até R$ 1,60

🎛️ 4. Filtros Avançados

O que é: Refine resultados de busca com múltiplos critérios.

Filtros disponíveis:
  • Categoria: Arroz, Feijão, Macarrão, Café, Leite, Óleo, Farinha, Suco
  • Mercado: Savegnago, Favetta, Pague Menos
  • Faixa de Preço: Digite mínimo e máximo desejado

💡 Exemplo: Busque "Arroz" + Categoria "Arroz" + Preço max "R$ 30" = resultados filtrados!

📈 5. Gráficos e Análises

O que é: Visualizações visuais dos dados de preços e economia.

Gráficos disponíveis (em Comparador):
  • Gráfico de Linhas: Comparação de preços entre mercados por produto
  • Gráfico de Barras: Economia por produto (qual produto mais economiza)
  • Gráfico de Pizza: Distribuição média de preços entre mercados

💡 Interatividade: Passe o mouse nos gráficos para ver valores exatos!

🗄️ 6. Base de Dados Populada

O que é: Banco de dados com produtos reais e preços variados entre mercados.

Dados incluídos:
  • 8 produtos em diferentes categorias
  • 3 mercados: Savegnago, Favetta, Pague Menos
  • Preços reais com variações entre lojas
  • Todos com disponibilidade marcada

📦 Produtos: Arroz, Feijão, Macarrão, Café, Leite, Óleo, Farinha, Suco

🚀 Fluxo Completo de Uso
  1. Busque um produto: Acesse Buscar e digite "Arroz"
  2. Veja as opções: Sistema mostra preços em 3 mercados diferentes
  3. Compare economia: Badge verde mostra "Economize até R$ X,XX"
  4. Adicione à lista: Clique "+ Adicionar à lista" no melhor preço
  5. Veja comparação geral: Acesse Comparador para análise visual
  6. Analise gráficos: Entenda tendências de preços entre mercados
  7. Organize compras: Acesse Minha Lista para finalizar
⚙️ Tecnologias Utilizadas
  • Frontend: HTML5, Bootstrap 5, JavaScript ES6
  • Gráficos: Chart.js (biblioteca de visualização)
  • Backend: PHP com PDO
  • Banco de Dados: MySQL/MariaDB
  • Armazenamento Local: LocalStorage para lista de compras
💡 Dicas Importantes:
  • A busca é case-insensitive e ignora acentos
  • Os filtros podem ser combinados para resultados mais precisos
  • Sua lista de compras é salva localmente no navegador
  • Os gráficos são responsivos e adaptam para mobile
  • Você pode adicionar novos produtos em "Adicionar Produto"
